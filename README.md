netiquette-checker
==================

## Getting Started

### Prerequisites

* Python >= 3.4

### Usage

```bash=
cd netiquette-checker/source/
python netiquette-checker.py SUBJECT FILE
```

*SUBJECT is the subject of the message you want to send.
FILE is the path to a file containing your message content.*
